# 조건문

[예제](./if.html)

## if 기본 문법

```
if (조건식) {
    // 조건식이 참(true)인 경우 실행될 구문
}
```

-   `{}`는 (실행될 구문이 한 줄인 경우) 생략 가능하지만 써 주는 것이 좋다.
-   **들여쓰기(indent)**를 보기 좋게 해야한다 (필수)

## if + else

```
if (조건식) {
    // 조건식이 참(true)인 경우 실행될 구문
} else {
    // 조건식이 거짓(false)인 경우 실행될 구문
}
```

### if + else if + else

```
if (조건식1) {
    // 조건식이 참(true)인 경우 실행될 구문
} else if (조건식2) {
    // 조건식1이 거짓(false)이고,
    // 조건식2가 참(true)일 경우 실행될 구문
} else {
    // 조건식1, 2가 모두 거짓일 경우 실행될 구문
}
```
